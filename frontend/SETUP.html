📊 YouTube Comment Sentiment Analysis - Setup Guide

Step 1: Backend Setup

Navigate to the backend directory and build the project:

cd backend mvn clean install

This will download all dependencies and compile the Java code.

Step 2: Run Backend Server

Start the Spring Boot application:

mvn spring-boot:run

Or if using Maven wrapper:

./mvnw spring-boot:run

Backend will start on http://localhost:8080

Step 3: Run Frontend

Open a new terminal and navigate to the frontend directory:

cd frontend

Then start a local server. Choose one:

Using Python:

python -m http.server 3000

Using Node.js:

npx http-server -p 3000

Or simply: Open index.html directly in your browser

✓ Ready to Use!

Open your browser and go to:

http://localhost:3000

Or if opening directly:

file:///path/to/youtube-sentiment-analysis/frontend/index.html

Testing the API

You can test the backend API using curl:

curl -X POST http://localhost:8080/api/sentiment/analyze \ -H "Content-Type: application/json" \ -d '{"videoUrl":"https://www.youtube.com/watch?v=dQw4w9WgXcQ","apiKey":""}'
